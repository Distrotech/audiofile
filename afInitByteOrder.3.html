afInitByteOrder(3) Manual Page

NAME

afInitByteOrder - initialize byte order for a track in an audio file setup

SYNOPSIS

#include <audiofile.h>
void afInitByteOrder(AFfilesetup setup, int track, int byteOrder);

DESCRIPTION

Given an AFfilesetup structure created with afNewFileSetup(3) and a track identified by track (AF_DEFAULT_TRACK for all currently supported file formats), afInitByteOrder initializes the track to the specified byte order.

PARAMETERS

setup is a valid file setup returned by afNewFileSetup(3).

track is always AF_DEFAULT_TRACK for all currently supported file formats.

byteOrder is one of AF_BYTEORDER_BIGENDIAN or AF_BYTEORDER_LITTLEENDIAN.

ERRORS

afInitByteOrder can produce the following errors:

AF_BAD_FILESETUP

setup represents an invalid file setup.

AF_BAD_TRACKID

track represents an invalid track identifier.

AF_BAD_BYTEORDER

byteOrder is not one of AF_BYTEORDER_BIGENDIAN or AF_BYTEORDER_LITTLEENDIAN.
