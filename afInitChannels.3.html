afInitChannels(3) Manual Page

NAME

afInitChannels - initialize number of channels for a track in an audio file setup

SYNOPSIS

#include <audiofile.h>
void afInitChannels(AFfilesetup setup, int track, int channels);

DESCRIPTION

Given an AFfilesetup structure created with afNewFileSetup(3) and a track identified by track (AF_DEFAULT_TRACK for all currently supported file formats), afInitChannels initializes the track to the specified number of channels.

ERRORS

afInitChannels can produce the following errors:

AF_BAD_FILESETUP

setup represents an invalid file setup.

AF_BAD_TRACK

track represents an invalid track identifier.

AF_BAD_CHANNELS

channels specifies an invalid number of channels.

SEE ALSO

afNewFileSetup(3), afInitSampleFormat(3)
